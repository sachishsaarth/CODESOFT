# CodSoft Data Science Internship

This repository contains the projects completed as part of the **CodSoft Data Science Internship**. Each project demonstrates different Machine Learning concepts, including classification and regression, using Python and Scikit-learn.

---

## 📂 Projects

### 🔹 Task 1 – Titanic Survival Prediction

Developed a Machine Learning classification model to predict whether a passenger survived the Titanic disaster based on passenger information.

**Highlights**
- Data Cleaning & Preprocessing
- Data Visualization
- Feature Encoding
- Random Forest Classifier
- Accuracy Evaluation
- Sample Prediction

📁 Folder:
```
Task1_Titanic_Survival_Prediction
```

---

### 🔹 Task 2 – Movie Rating Prediction with Python

Built regression models to predict movie ratings using features such as genre, director, actors, year, duration, and votes.

**Highlights**
- Data Cleaning
- Feature Encoding
- Linear Regression
- Random Forest Regressor
- Model Comparison
- Movie Rating Prediction

📁 Folder:
```
Task2_Movie_Rating_Prediction
```

---

### 🔹 Task 3 – Iris Flower Classification

Developed a Machine Learning classification model to classify Iris flowers into their respective species using flower measurements.

**Highlights**
- Data Visualization
- Feature Encoding
- Random Forest Classifier
- Accuracy Evaluation
- Sample Prediction

📁 Folder:
```
Task3_Iris_Flower_Classification
```
