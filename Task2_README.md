# Movie Rating Prediction with Python

## CodSoft Data Science Internship - Task 2

This project predicts the rating of a movie based on features such as genre, director, actors, year, duration, and votes using Machine Learning regression techniques.

## Objective

The objective of this project is to build a regression model that estimates movie ratings by analyzing historical movie data and identifying patterns that influence ratings.

## Dataset

- **Dataset:** IMDb Movies India Dataset
- **File:** `IMDb Movies India.csv`

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

- Import Libraries
- Load Dataset
- Data Exploration
- Data Preprocessing
- Data Cleaning
- Feature Encoding
- Train-Test Split
- Model Training
- Model Evaluation
- Sample Prediction

## Machine Learning Models

- Linear Regression
- Random Forest Regressor

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Results

Both Linear Regression and Random Forest Regressor were trained and evaluated.

The **Random Forest Regressor** achieved better performance.

- **Mean Absolute Error (MAE):** 0.85
- **Root Mean Squared Error (RMSE):** 1.13
- **R² Score:** 0.31
