# Iris Flower Classification

## CodSoft Data Science Internship - Task 3

This project classifies Iris flowers into three different species using Machine Learning classification techniques based on sepal and petal measurements.

## Objective

The objective of this project is to build a classification model that accurately identifies the species of an Iris flower using its sepal length, sepal width, petal length, and petal width.

## Dataset

- **Dataset:** Iris Dataset
- **File:** `IRIS.csv`
